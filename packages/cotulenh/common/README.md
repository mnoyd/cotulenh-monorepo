# @cotulenh/common

Shared utilities, constants, and error handling for the CoTuLenh monorepo.

## Installation

```bash
pnpm add @cotulenh/common
```

## Features

- **Centralized Logging**: Consistent logger interface across client and server.
- **Error Handling**: Standardized `ErrorCode` and `CoTuLenhError` classes.
- **Constants**: Shared board dimensions, color definitions, and piece symbols.
- **ESLint/TSConfig**: Shared development configurations.
