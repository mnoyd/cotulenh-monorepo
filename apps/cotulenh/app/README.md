# @cotulenh/app

The official web interface for CoTuLenh (Commander Chess). Built with SvelteKit and Vite, providing a modern warfare-themed interactive game experience.

## Installation

```bash
pnpm install
pnpm dev
```

## Features

- **Interactive Board**: Full implementation of Commander Chess rules and movement.
- **Move History**: Complete SAN-based move history and navigation.
- **Deployment System**: Specialized UI for piece stacking and combination.
- **Visual Influence**: Real-time visualization of Air Defense zones.
- **Responsive Design**: Optimized for both desktop and mobile play.

## Tech Stack

- **Framework**: SvelteKit 2
- **Styling**: Tailwind CSS 4
- **State Management**: Svelte Stores + Immer
- **Logic**: Powered by `@cotulenh/core` and `@cotulenh/board`
