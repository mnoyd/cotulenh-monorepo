# Welcome to Cotulenh: Basic Movement

In Cotulenh, each unit type has unique movement characteristics. Understanding these patterns is fundamental to mastering the game.

## What You'll Learn

This subject covers all 11 piece types across three progressive sections:

### 1. Basic Units

**Infantry, Engineer, Militia, Commander, and Headquarters**

- Master the foundation of unit movement
- Learn the differences between orthogonal and all-direction movement

### 2. Medium Range Units

**Tanks, Anti-Air, and Missiles**

- Expand your tactical options with 2-3 square ranges
- Discover the unique circular pattern of Missiles

### 3. Advanced Units

**Artillery, Air Force, and Navy**

- Harness unlimited range with Artillery
- Master the agile mobility of Air Force
- Control the seas with Navy

## The Game Board

The board is **11 files × 12 ranks**:

| Dimension | Range    | Description                |
| --------- | -------- | -------------------------- |
| Files     | a - k    | Left to right (11 columns) |
| Ranks     | 1 - 12   | Bottom to top (12 rows)    |
| River     | Rank 6-7 | Divides the board          |

## Movement Types at a Glance

| Unit         | Range | Pattern                             |
| ------------ | ----- | ----------------------------------- |
| Infantry     | 1     | Orthogonal                          |
| Engineer     | 1     | Orthogonal                          |
| Militia      | 1     | All 8 directions                    |
| Tank         | 2     | Orthogonal                          |
| Anti-Air     | 1     | Orthogonal                          |
| Missile      | 2     | Circular (2 straight or 1 diagonal) |
| Artillery    | ∞     | Orthogonal                          |
| Air Force    | 4     | Any direction (flies)               |
| Navy         | 4     | All directions (water only)         |
| Commander    | ∞     | Orthogonal                          |
| Headquarters | 0     | Immobile                            |
