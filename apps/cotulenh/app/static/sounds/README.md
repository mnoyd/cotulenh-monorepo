# Sound Effects

This directory contains sound effect files for the game.

## Required Files

- `move.mp3` - Standard piece movement
- `capture.mp3` - Piece capture
- `check.mp3` - Check notification
- `game-end.mp3` - Game over (checkmate, stalemate, draw)
- `error.mp3` - Invalid action
- `deploy.mp3` - Deploy/stack action complete

## Recommended Sources

Free sound effects can be found at:

- [Lichess sounds](https://github.com/lichess-org/lila/tree/master/public/sound) (MIT License)
- [Freesound](https://freesound.org/) (check individual licenses)
- [Zapsplat](https://www.zapsplat.com/) (free with attribution)

## Audio Format

MP3 format is recommended for browser compatibility. Keep files small (< 50KB each) for fast loading.
