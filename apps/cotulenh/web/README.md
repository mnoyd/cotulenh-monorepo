# @cotulenh/web

Next.js web application for CoTuLenh, integrated in the monorepo.

## Development

From repo root:

```bash
pnpm dev --filter @cotulenh/web
```

Build:

```bash
pnpm build --filter @cotulenh/web
```

Lint:

```bash
pnpm --filter @cotulenh/web run lint
```

Type-check:

```bash
pnpm --filter @cotulenh/web run check-types
```
